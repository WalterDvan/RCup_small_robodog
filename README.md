# RCup_small_robodog
中国机器人大赛暨RoboCup机器人世界杯中国赛 —— **四足仿生机器人·快递运送赛（小型组）**

---

## 一、赛项信息
- 赛事：中国机器人大赛暨RoboCup机器人世界杯中国赛
- 赛项：四足仿生机器人 - 快递运送赛（小型组）
- 比赛时间：2026年10月（石家庄总决赛）
- 模式：全自主运行，无遥控

## 二、比赛任务
四足仿生机器人从快递集散中心出发，依次通过：
减速带 → 上下斜坡 → 分岔路 → 警戒区 → 上下高台
识别颜色标识，将**棕色/紫色快递球**投递至对应住户区，返回集散中心完成任务。
正式比赛共两轮：棕色快递一轮、紫色快递一轮。

## 八、Git 使用说明（自用）

对网络一般要求是要有梯子（watt toolkit这个软件即可，这是个电脑的软件），但是我用校园网也可以连上github，不论是宿舍的还是教学楼的，所以有校园网的话梯子也不是必备的。

## 1. 第一次把仓库下载到电脑（**克隆**）
打开 **PowerShell 或 Git Bash**，进入你想放的文件夹，执行：
```bash
git clone https://github.com/WalterDvan/RCup_small_robodog.git
```

## 2. 每次开始工作前，先更新仓库（**拉取**最新内容）
```bash
git pull
```

## 3. 修改/新增文件后，**上传**到 GitHub
```bash
git add .
git commit -m "这里写你做了什么修改"
git push
```
